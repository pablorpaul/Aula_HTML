O que é HTML?

HTML é uma linguagem de marcação utilizada para estruturar páginas da web. Ele define títulos, parágrafos, imagens, links, etc.

O que é CSS?

CSS é uma linguagem utilizada para estilizar páginas HTML. Ele conrola cores, fontes, espaçamentos e o layout dos elementos.
